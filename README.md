 Clínica Olho Feliz
Site institucional de uma clínica oftalmológica fictícia, desenvolvido como exercício prático de HTML.
 Páginas

index.html — Página principal com informações de contato e mapa

about.html — Sobre a clínica

hours.html — Tabela de horários de atendimento

contact.html — Formulário de contato

 Tecnologias

HTML5

 Objetivo
Projeto acadêmico para praticar estrutura de páginas HTML com navegação entre múltiplas páginas, formulários, tabelas e iframe do Google Maps.
